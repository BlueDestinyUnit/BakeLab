### 내가 수행한 역할

- 상품 등록 기능 구현

- 검색 기능 구현

### 프로젝트 상태
- 기본적으로 구매자만 회원가입이 가능합니다.
- 판매자 계정에 대한 UI 보완이 필요합니다.
- 검색 기능에 일부 버그가 존재합니다.
- 회원은 장바구니 기능을 정상적으로 사용할 수 있지만, 비회원은 아직 장바구니 기능에 버그가 있습니다.
- 메뉴바 UI를 개선해야 합니다.
